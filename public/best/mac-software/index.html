Mac Software

Mac OS X Mavericks is my main operating system now. I still use Windows and Linux a bit, but by far this is where I do most of my work.

I also have a pages for the Best Windows Software and Best Software Development Tools.

For an always up to date list, check out my Best Software List on Github.

Cloud Storage, Sync, and Backup

  • Dropbox - Service that keeps your files safe, synced, and easy to share.
  • Drobo - Get the software, after the hardware.
  • Google Drive - Google’s cloud service tied to your Gmail.
  • OneDrive - Microsoft’s cloud service.

Internet

  • Chrome - Google web browser.
  • Firefox - My second favorite browser.
  • Transmission - Torrent downloader.
  • Tweetdeck - Twitter client with columns.

Miscellaneous and Utilities

  • Adobe Acrobat Reader - PDF reader and editor.
  • Android File Transfer - Transfer files via USB to your Android device.
  • Calibre - One stop solution to e-book needs.
  • Filezilla - SFTP client.
  • Karabiner - A powerful keyboard customizer.
  • GPG Suite - Secure way to encrypt communication.
  • Paw (HTTP & REST Client) - Helps you build HTTP requests, inspect the server’s response and even generate client code.
  • Seil - Utility to remap Caps lock and other keys.
  • SuperDuper - Clone hard drives.
  • Teamviewer - Remotely control another computer.
  • Trim Enabler - Enables TRIM on 3rd party SSDs.
  • Unarchiver - Decompress zip/rar files.
  • World Community Grid - Grid computing.

Audio and Video

  • Handbrake - Edit/convert videos.
  • Livestreamer - Command-line utility that extracts streams from various services and pipes them into a video player of choice.
  • Noizio - App that creates ambient sounds of coffee shop, rain, ocean, etc.
  • Plex Media Server - Media server for your home videos/photos.
  • SoundNode - Soundcloud desktop app.
  • Spotify - Music player.
  • Spotifree - Mute ads in Spotify.
  • Tagger - Batch update your ID3 tags in your music collection (mp3s).
  • VLC Media Player - The best video/media player for any OS.

Pictures and Images

  • Gimp - Edit pictures, a Photoshop-like program that’s free.
  • Google Photos - Google’s photo storage.
  • Picasa - Google’s older photo storage service.

Productivity

  • Alfred - Type what you need or what you want to do.
  • F.lux - Automatically adjust your computer screen to match lighting.
  • Glances - System monitoring tool that runs in terminal.
  • Skitch - Edit your screenshot images.
  • Slack - Collaboration software for teams.

Best Applications to Buy

  • 1Password - Password Manager and Secure Wallet.
  • Adobe Creative Cloud - Photoshop + Lightroom CC. Well worth the $10/month subscription.
  • Airmail 2 - Mail client that’s better than default.
  • Bartender 2 - Organize your menu bar apps.
  • Better Touch Tool - Custom map your keys.
  • Divvy Window Manager - Window management for tiling your windows.
  • Forecast Bar - Weather in taskbar.
  • ForkLift2 - Best Finder replacement.
  • iStat Menus - System monitor for CPU, memory, HDD, battery, etc.
  • iStat Mini - Mini version of iStat Menus.
  • Parallels - Run other OSes as virtual machines.
  • Stock+ Pro - Monitor stocks.
  • Zooom/2 - Resize your windows without having to click on corners.
