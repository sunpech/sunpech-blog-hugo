My Favorite Firefox Add-ons

Posted on November 23, 2008 by Steven Suwatanapongched
 Technology

Firefox is my favorite web browser. In this day and age, I see no need to use Microsoft’s Internet Explorer (IE). Although I do like Google Chrome, it’s still not available for Linux yet. But this isn’t an entry about browsers and the pros and cons of each. (I do plan on writing about browsers and operating systems in the near future.)

This is the list of add-ons that I install on any computer I use running Firefox.

  • Adblock Plus - Blocks Ads (a must for anyone serious about web browsing)
  • Forecastfox - Weather monitor
  • Delicious Bookmarks - Social Bookmarking
  • Fast Dial - Nice thumbnail size shortcuts to websites you choose
  • Server Spy - Shows what kind of server is running on the page I'm viewing
  • ShowIP - Shows the IP address of the site I'm viewing
